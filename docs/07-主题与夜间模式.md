# 主题与夜间模式 & 配色规范

## 架构

```
SettingsProvider.themeMode
    ↓
MyApp (Builder + context.watch)
    ↓
MaterialApp.router(theme / darkTheme / themeMode)
    ↓
ThemeData(colorScheme: ColorScheme.fromSeed(seedColor: ...))
    ↓
所有 Widget 通过 Theme.of(context).colorScheme 获取语义色
```

- 主题模式由 `SettingsProvider._themeMode` 控制，持久化到 `SharedPreferences`
- `main.dart` 的 `Builder` 通过 `context.watch<SettingsProvider>()` 实时监听变化
- `MaterialApp.router` 同时提供 `theme`（浅色）和 `darkTheme`（深色），通过 `themeMode` 切换
- 两个主题都基于 `ColorScheme.fromSeed(seedColor: ...)` 自动生成

## 配色核心原则

### 分层策略

```
大布局/背景  →  ColorScheme 语义色（自动跟随主题）
  └─ Card、ListTile、Section 背景 → surface / surfaceContainerLow
  └─ 分割线、边框 → outlineVariant / Divider()
  └─ 正文、副文 → onSurface / onSurfaceVariant

小部件/图标  →  固定色（不随主题变化）
  └─ 功能图标（设置页、用户空间）→ 预设色板
  └─ 等级/用户组标签 → #FF9900
  └─ 板块标签 → #53BCF5
  └─ 操作标识 → 语义明确的固定色
```

### 原则

1. **布局中性化** — 页面背景、卡片、容器、列表项全部使用 `colorScheme` 语义色，确保深色模式自动适配
2. **小部件写死** — 图标、标签、badge 使用固定颜色，不跟随主题色（避免主题切换后颜色变淡/变突兀）
3. **对比度优先** — 固定色搭配 12% 透明度的背景，确保浅色/深色模式下都有足够对比
4. **语义明确** — 同类型功能使用相同颜色，避免混淆

## 大布局颜色映射

| 场景 | 正确写法 |
|------|---------|
| **页面背景、Card 背景、AppBar 背景** | `colorScheme.surface` |
| **浅色背景区**（空状态、次要容器） | `colorScheme.surfaceContainerLow` |
| **中等背景区**（表头、选中态） | `colorScheme.surfaceContainerHigh` |
| **边框、分隔线** | `colorScheme.outlineVariant` |
| **主文本** | `colorScheme.onSurface` |
| **次要文本、辅助图标** | `colorScheme.onSurfaceVariant` |
| **高亮色**（FilledButton、选中态） | `colorScheme.primary` / `colorScheme.primaryContainer` |
| **错误/删除/危险** | `colorScheme.error` / `colorScheme.errorContainer` |

### 常规用法模板

```dart
final cs = Theme.of(context).colorScheme;

Container(
  color: cs.surface,
  child: Column(
    children: [
      Text('标题', style: TextStyle(color: cs.onSurface)),
      Text('副标题', style: TextStyle(color: cs.onSurfaceVariant)),
      Icon(Icons.star, color: cs.primary),
    ],
  ),
)
```

### 分割线

**永远不要**给 `Divider` 或 `VerticalDivider` 指定颜色：

```dart
// ❌ 错误
Divider(color: Colors.grey.shade200)
Divider(color: Colors.grey.shade300)

// ✅ 正确
Divider()
VerticalDivider()
```

## 小部件固定色板

以下颜色直接写死，不通过 `colorScheme` 获取，不随主题色变化。

### 1. 功能图标色板（设置页、搜索页）

每个图标分配独立颜色，搭配 12% 透明度圆形背景：

| 色值 | 用途 | 示例图标 |
|------|------|---------|
| `#2196F3` 蓝 | 站点/网络相关 | `dns`, `language` |
| `#4CAF50` 绿 | 用户/正向操作 | `person`, `thumb_up_outlined` |
| `#FF9800` 橙 | 内容/主题 | `forum`, `palette`, `monetization_on_outlined` |
| `#FFC107` 琥珀 | 娱乐/表情/金币 | `emoji_emotions`, `workspace_premium_outlined` |
| `#9C27B0` 紫 | 链接/快捷 | `link` |
| `#607D8B` 蓝灰 | 历史/设置 | `history`, `settings` |
| `#795548` 棕 | 存储/数据 | `storage` |
| `#E91E63` 粉 | 计算/好友 | `calculate`, `people_outline` |
| `#FF5722` 深橙 | 装饰/分享 | `palette_outlined`, `share_outlined` |
| `#3F51B5` 靛蓝 | 界面/选项卡 | `tab` |
| `#00BCD4` 青 | 搜索/回帖 | `search`, `reply_outlined` |
| `#E65100` 深橙 | 操作警告 | 小黑屋禁止发言 |

```dart
// _iconBox 工具组件（设置页）
Widget _iconBox(IconData icon, Color color) {
  return Container(
    width: 44, height: 44,
    decoration: BoxDecoration(
      color: color.withValues(alpha: 0.12),
      borderRadius: BorderRadius.circular(12),
    ),
    child: Icon(icon, color: color),
  );
}

// 调用示例
leading: _iconBox(Icons.person, const Color(0xFF4CAF50)),
```

### 2. 标签/Badge 色板

| 场景 | 文字色 | 背景色 | 说明 |
|------|--------|--------|------|
| **用户等级** | `#FF9900` | `#FF9900` + 12% 透明度 | 全局统一，不区分用户组 |
| **板块标签** | `#53BCF5` | `#53BCF5` + 12% 透明度 | 帖子卡片中的板块名 |
| **在线状态** | `#4CAF50` | `#4CAF50` + 12% 透明度 | 用户空间页 |
| **楼主标识** | `onSurfaceVariant` | `surfaceContainerLow` | 主帖标识，中性低调 |
| **楼层标签** | `onSurfaceVariant` | `surfaceContainerLow` | 楼层号 |

```dart
// 用户等级示例
Container(

  ),

    style: const TextStyle(fontSize: 11, color: Color(0xFFFF9900)),
  ),
)
```

## 特殊情况

### 图片查看器（GalleryViewer）

背景保持 `Colors.black`，全屏沉浸式沉浸不应跟随主题。

### BBCode 颜色映射表

`post_ast_widget.dart` 中的 `_colorNameToFlutter` 映射表是功能性映射，不属于样式颜色。

### WebView

`flutter_inappwebview` 加载的论坛网页不受 Flutter 主题控制。

## 验证

新增 UI 后，在三种模式下检查：

1. **浅色模式**（设置 → 夜间模式 → 浅色）
2. **深色模式**（设置 → 夜间模式 → 深色）
3. **跟随系统**

确保：
- 大布局颜色自动跟随主题，无突兀色块
- 固定色图标/标签在深色模式下仍有足够对比度
- 分割线可见但不过分刺眼
- 同语义元素颜色统一
