# 概述与架构

## 项目定位

Discuz + MT 论坛（克米模板）的 Flutter 客户端。Discuz 无官方 API，通过 `lib/api/` 层解析 HTML/XML 为结构化 JSON。

## 分层架构

```
lib/
  api/        — HTTP 请求 + HTML/XML 解析，三文件规范（http / parse / export），test 统一放在 test/api/
  services/   — 无状态业务逻辑（ApiService 管理 Dio/MtImageHosting/cache 等）
  providers/  — 有状态 UI 数据（ChangeNotifier + Provider）
  core/       — 通用工具（BBCode 解析/渲染、ShortcutHelper、Logger 等）
  widgets/    — 可复用组件（BBCodeToolbar/ImagePickerSheet/GalleryViewer 等）
  pages/      — 页面级 Widget，按功能拆文件（>500 行时拆分）
  config/     — 全局配置（路由/工具栏/站点配置）
  models/     — 纯数据模型
  auth/       — 登录态管理
```

## 渲染管线

```
Discuz 服务器 → API HTTP (Dio) → HTML/XML → parse 层 → 结构化数据 → Flutter Widget
                                                        ↓
                                              BBCode → BBCodeParser → AST → Widget
```

## 路由

GoRouter 实现，`lib/config/router.dart` 集中定义。AppShell 包裹底部导航（首页/导读/社区/我的），设置页嵌套子路由。

## 核心工具组件

| 文件 | 职责 |
|------|------|
| `core/bbcode_parser.dart` | BBCode → AST 解析器 |
| `core/html2bbcode.dart` | HTML → BBCode 逆向转换 |
| `core/page_fetcher.dart` | 通用论坛页面表单数据提取 |
| `core/xml_helper.dart` | XML 解析工具 |
| `core/logger.dart` | `AppLogger` 统一日志 |
| `core/shortcut_helper.dart` | 快捷键序列化解析 |
